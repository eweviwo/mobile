# Mobile App

This repository will develop a mobile application for the project. It will be in Android and iOS version. 

## Concept
- The mobile app will serve content (podcasts) as it is presented on the site's web. 
- The mobile app will share resourceful links for GitHub contribution. 
- The mobile app will share a link to our donation page.
- The mobile app will allow users to share to others on their devices. 

## Development

- Preferred programming language of choice will be [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) and [TypeScript](https://www.typescriptlang.org/). 
- Preferred cross-platform framework of choice will be [React Native](https://reactnative.dev/).
- Preferred supporting framework will be [Expo](https://expo.dev/).
